Supported Formats

Bases 2-36: Use digits 0-9 and letters A-Z
Base 60 (Babylonian): Use comma-separated values (0-59)
Example: 1,30,45 represents 1×60² + 30×60¹ + 45×60⁰
Decimals: Use dot (.) for fractional parts in bases 2-36
